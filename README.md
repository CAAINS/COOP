## 希望的技术

在1945年，凯恩斯从《经济学杂志》主编卸任，顾问委员会为其举办了离别宴会。致谢时，他对全场的皇家经济学会和经济学家干杯，他说在座的都是受托人(to trustee ....)，经济学家们以为说的是自己，他停顿了一下后继续说："为经济学家、为并非创造文明之受托人，而是创造文明之可能的受托人诸位，干杯。" 显然凯恩斯认为真正的文明创造者是那帮他的艺术家和作家朋友们。或許，以前经济学家与艺术工作者还真是创造文明的受托人，现在则是网络世界赢者全拿的菁英们了。

要有自主的数字生活，再度成为创造文明的受托人，就要把技术生产资料的社会使用权赢回来，这包括技术资料和技术对象。技术资料是指软件、硬件、算法和媒介；技术对象是用户的隐私和人类一生中的所有足迹。社会使用，则是指这种技术生产资料的社会化，也就是应该更民主地、更公开地、更平等地共享生产价值。我们不是反对技术资料，而是反对把它们作为剥削的手段来使用。（参考黄孙权于2019于加拿大多伦多大学亚洲中心的演讲稿，[TAKING ROOTS: CODING & DESIGN FOR PLATFORM CO-OPS](https://hackmd.io/yOZ84lZXTCyxps_YxVUXKA?view).）

`INS` 发行 social token，（下称`$COOP`，我们更愿意翻译成社会通证），我们称之为一种"**希望的技术**"，通过加密技术、区块链信任与合作精神，由下而上的发展成一个涵盖学术、科技与艺术领域的合作社群。平台合作主義的倡議者Trebor Scholz和Nathan Schneider 共同编辑的，***去骇回我們擁有的***（Ours to Hack and to Own）一書中有许多成功案例了。这里，我们将以不同的方式，同步思考技术和社会需要来打造当代的的[罗需戴尔公平先锋社](https://en.wikipedia.org/wiki/Rochdale_Society_of_Equitable_Pioneers)，用新技術與社群力量生產自己精神所需的粮食杂货之"产销合作社"，以1844年先锋社所创立的[罗需戴尔原则](https://zh.wikipedia.org/wiki/%E7%BD%97%E8%99%9A%E6%88%B4%E5%B0%94%E5%8E%9F%E5%88%99)─这也是国际合作社联盟遵循至今的原则─来建立社群。我们的希望是是：

* 首先，**让点对点的交换成为可能**。比特币的创始者中本聪在白皮书中的梦想是让加密货币点对点的交易像日常生活花零钱一样便利，然而当下除了兑换法币与私下交易，人们无法使用加密货币买杯咖啡或一本书，交换时间与财货。要过上无银行（bankless）生活，就要使人与人的交易像发 email 一样简单，需要信任的群体与信任的技术。
* 其次，**连结生产与消费**。区块链与加密货币的原初为的是解决人们生活困局，而非创造飞地。然而，仅仅改变流通工具的效率与便利，是不能改变与之相关连的生产和消费的。普鲁东主义者，1990年代流行的社区货币，或者2001，2002阿根廷发行的coupon(就是token,也就维持了两年)都曾经有过以失败告终的经验，当今的加密通证技术若不改变其视野与社会性构造的方式，就仅仅只是又一个历史重复而已。从2008的经济危机后获得生命的加密货币，逐渐成为狂噪的金融游戏：投资、期权、合约对战、杠杆各种衍生品，在流通领域疯狂，在交易所上热络，但对实际的生产消费影响甚微，这些情境与2008年金融危机前次贷泡沫景象一模一样。我们必须将区块链与加密签署等流通技术，透过社群与合作经济模式，改变生产和消费模式。
* 第三；**重建社群中的生产关系**。网络上的内容生产者通常为零工经济的最下层，除了少数的up主、网红，付费墙内的知名人士，内容生产者是辛勤（或上瘾）是无工资的。平台与中介组织的利润机制创造了一种新职位：产销者（Prosumer），产销者首先是消费者，藉由消费内容来生产更多的的消费资料。产销者越积极，平台赚取的利润就越多，这美其名"财富的革命"，是借由信息时代的监控与行为预测，广告与数据，导致意识的统一化，工业化，导致个性的消失，这形成了剧烈内卷化的网络技術与个体的集体化。即便社交媒体现在积极地采取小费制、点赞、资助众筹、付费墙等设计来弥补无偿劳动，或说创造更多的产销者，但这远远不够的。劳动价值体现视特定社会必要劳动时间的长短，但劳动价值的榨取却瞬秒间可完成。我们都在帮脸书、微信、微博、推特、Instagram、抖音打工。《21世纪资本论》指出资本获利已经远远超过工资成长率，这是贫富拉开差距的主要原因。任何”自主”劳动若是意味生产者能够获得社会需要决定之劳动价值，就只能在社群中形成生产与消费，而非在个体与平台之间，即，重建社群内的生产关系，联盟化生产与消费链条是拒绝成为異化的"产销者"的可行方法之一。
* 第四、**建立新生活**。资本累积的形式有两个隐藏成本的方式，擢取式积累(salvage accumulation)是利用土著知识获取利润，如早期殖民利用原住民的当地知识帮忙猎鲸、采集松茸；或如现在的社交媒体利用使用者生产内容而赚取暴利。而边缘资本主义(pericapittalist)则是不用控制生产过程，却能借由过程累积财富的新形态，将系统外的区块生产转化成自身的利益。如现代化的工业生产贡献不计废弃物与污染环境的外部成本；如当今炒作"加密艺术"而不需要负担艺术养成的成本；將科技进步仅仅视为个人/小群体的创新，忽略政策补贴与税租优惠，不计算公共投资如教育、基础设施的投入等等。我们必须反对不計算”外部性”成本的财富积累方式。如奈格里与哈特在 ***Commonwealth*** 中再三强调的，外部性就是我们的生活，我们的都会，我们的地球。外部性就是我们的社会的再生产条件，是人們生活的整體。

我们要将颠倒的世界颠倒过来，我们从合作社能学到的比更多金融专家告诉我们的更多。今天，合作经济在世界都有长足的贡献，成为共同富裕的动力，如日本消費者合作社聯盟為全國31%的家庭提供服務，巴西40%的農業、丹麥36%的零售市場都是由合作社組成的。肯亞45%、紐西蘭22%的國內生產總值都來自合作社。英國目前有20萬人在400多家合作社工作。在德國，柏林市民們目前形成公共事業合作社來購買和運營城市電網，有数量足够的多元的、跨年龄与种族的合作住宅等。而新兴的平台合作社已经在艺术、农业、交通、软体、教育、零售取得非常好的成绩。总之，区块链的本质就是合作社的本质，新技術已经准备好了，有待创新地使用。我们以四个原则来表达希望的技术之内容：

1. **自愿与开放的社員制**：无中心的社群信任是加密货币的精神。但加密货币若只剩下金融投资与买卖，我们将只拥有帐本而没有生活。Web 3.0 的充分条件是如加密、区块链、去中心身份与无需信任的交易等等，但必要条件是人们使用的场景，是技术传播的过程是否基于使用价值而产生的交换经济体系。通过社會通證，可以由下而上的建立社群信任的经济交换系统。
2. **自主與自立**：我们需要从个人和组织的一次性捐赠转变为对公共产品做出长期承诺的整个机制。不需要慈善机构与社会企业修补的公平机制。从人类学者David Greaber的观点来看，虚拟货币（信贷记帐）早于货币出现，而以物易物则是货币不足或崩溃才产生的，这点在2008年后[全球的另类经济体系](https://www.caa-ins.org/archives/1909)蓬勃发展更清晰可见。在法国人类学家莫斯的经典研究《礼物》中，以物易物是群體間的集體節慶活動，而非個人在自由市場中找尋要兌換之物。古典经济学与新自由主义经济学假设经济理性个体参与自由市場的假设几乎破产，现实与理论都是。货币从来不是人类的交易的原初形式，信贷的记帐才是，即，加密技术通证（在古代，就是放在上锁的柜子里有众多签名的纸张）才是。在小部落村庄里的信任，通过加密技术和区块链已然可以跨地理来实现无需信任的信任(trustless)。`$INS`旨在创造此种社群内外"缺失"的信任，以建立在智能合约上的信任而能够自由交易、协同、互助来形成社群。我们已经见到许多 NFT 场景与 DAO 所展示的潜能。我们更须进一步，让新技术不仅成为展示财富快速累积的头像，而是成为创造共同富裕的动力，这乃自主与自立在數字社會的新意涵。
3. **教育、培训与信息**：INS所欲的社群，乃是一个结合学术、艺术、科技的社群，希望能够为其社员及"公众"提供教育和培训。这也是INS历年来举办公开的年会、讲座、开放课程、工作坊之意。在ICA发布的《合作社特征声明》里也提到“合作社应向其社员提供教育和培训，以便他们可以有效地为合作社的发展效力。合作社应向公众——尤其是年轻人及意见领导——提供关于合作的本质和优势方面的信息。”这也是我们的目标。
4.  **社间合作与关心社区**：你可以用以下的词汇，特别是它所代表的价值与愿景来形容我们正在创造的新世界：[时间银行](https://en.wikipedia.org/wiki/Time-based_currency)、[社区货币](https://www.seinsights.asia/specialfeature/5065/5092)、[礼物经济](https://en.wikipedia.org/wiki/Gift_economy)、[合作社](https://en.wikipedia.org/wiki/Cooperative)、[互助社](https://en.wikipedia.org/wiki/Credit_union)、[平台合作主义](https://platformhk.coop/)。我们希望自己只是这个大运动的同行者，愿意成为数字合作经济的棋手，分享、参与与共同创造。我们也相信，所有拥有`$COOP` 的人，越努力的发展自己的梦想，社群就会越走向他/她心之向往之地。
    

## 技術架構
`$COOP` 是發行在 [Nervos Network](https://www.nervos.org/) 上的 social token。Nervos Network 是一個开放的数字经济网络，通过新颖的設計結構解决区块链技术大规模应用的问题。Nervos Network 是一个分层架构网络，一层公链 CKB 关注保持去中心化与安全，二层扩容协议提供更好的性能与更灵活的场景适配。Nervos Network 通过抽象灵活的编程模型创造与互联网兼容的帐户体系，实现与其他区块链网络的互操作，降低用户的使用门槛。Nervos Network 与互联网类似，运行于一系列开放协议上，任何人都可以自由参与运行节点。

`$COOP` 遵循 Nervos Network 上 sUDT token standard 发行，能够无缝集成于 Nervos Network 生态，在所有支持该标准的应用中使用。`$COOP` 客户端是一个去中心化应用，基于去中心化私钥管理技术 Unipass 设计，运行在用户本地浏览器环境中，支持用户使用 Email 作为帐户管理自己的 `$COOP`  token，避免了不必要的使用门槛，無須複雜的操作就能够满足日常需要。每一个 `$COOP` 用户帐户同时也是 Nervos Network 上的去中心化帐户，需要占用 Nervos Network 公链的存储空间（CKByte）。通过参与 INS 活动或是为 INS 社群做出贡献获取到`$COOP` token 的用户（参考“获取”一节），其帐户所需的存储空间将在设立帐户时由 INS 免费提供。其他用户如果需要获得`$COOP` token，需要`$COOP` token 的持有者为其提供所需的存储空间（CKByte）。

第一阶段将实现`$COOP` token 的基本功能，满足社群经济的基本需求。`$COOP` token 是一种真正由用户持有，完全属于个人的 token，无论是 INS 社群还是其他任何人无法干涉用户对自己 `$COOP` token 的运用。未来我们可以根据发展需要，围绕 `$COOP` token 增加更多的功能和应用场景。

`$COOP` 为定量通证，发行上限为2300万枚，第一年首发行100万枚，第二年按照任务总量与贡献度共发行200万枚，其中百分之三十，按照拥有`$COOP` 的数量按比例以平方根的方式发送，作为社群贡献度的回馈。往后每年依据`$COOP` 地址数量的倍数发行，由社群治理共同决定发行量、回馈比例与公益回馈项目。

我们的社会通证技术是开源的，如今发行系统与社会钱包都已经公开的部署，希望有更多人的一同参与、改进。

## `$COOP` 的权利与使用场景
拥有` $COOP`者，即意味了参加了以 `INS` 为核心的学术、艺术家与技术团队共组的合作社，拥有准社员资格。当你贡献度达一定标准后（如拥有INS的NFT），则是社员，可以参与投票与组织推进决策，社员可以是个人或者组织。与一般公司股东制度跟DAO最大的不同是，社员无论拥有资产多少，投票权利是相等的，一人一票，而社员与准社员皆能以劳力时间或金钱投入来获取`$INS` ，此为对社群贡献度计算，每年度将会有适当的比例回馈。`INS` 将发行首批社员标志88枚NFT，其中有十六枚保留给联合发起者。使用 `$COOP` 或其他加密货币购买(比率為80：20，前者為奉獻度，後者為投資與贊助）。此 NFT 也是这个学术社群合作社的社员证，社员将可参与 `$COOP`的组织决策。此 NFT 可以转卖交易，社员资格可以转移，以拥有 NFT 为凭证。
